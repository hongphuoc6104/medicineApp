# Docs — Tài Liệu

## Nguồn active hiện tại

Các file nên đọc trước khi làm việc với sản phẩm hiện tại:

| File | Vai trò |
|------|---------|
| `../AGENTS.md` | Bối cảnh repo, guardrails kỹ thuật, map kiến trúc |
| `../APP_ACTIVE_GENERAL_PLAN.md` | Kế hoạch phát triển active duy nhất cho app/product |
| `../APP_ACTIVE_DETAILED_PLAN.md` | Quy tắc thực thi, workstreams, luật dừng |
| `UONG_THUOC_FULL_TEST_PLAN.md` | Release gate và test matrix cho flow chính |
| `phase_a_debug_runbook.md` | Runbook debug khi đụng app-path scan/contract |

## Tài liệu kiến trúc và trạng thái

| File | Nội dung |
|------|----------|
| `MASTER_PLAN.md` | Kiến trúc và trạng thái phát triển ở mức hệ thống; không phải execution plan active |
| `project_status.md` | Trạng thái tổng quát: đã làm, đang có, chưa làm |
| `phase_a_app_scan_alignment_completed.md` | Ghi nhận batch hoàn thành cho app-path scan |
| `phase_b_dose_verification_completed.md` | Ghi nhận batch hoàn thành cho Phase B MVP-lite |

## Tài liệu lịch sử hoặc tham khảo

| File | Nội dung |
|------|----------|
| `master_plan.md` | Kế hoạch cũ để tham khảo lịch sử kiến trúc, không phải nguồn active |
| `phase_b_mvp_lite_plan.md` | Plan chuyên đề cũ cho Phase B MVP-lite |
| `phobert_ner_plan.md` | Plan chuyên đề cho NER, chỉ đọc khi đụng training/model |
| `COLAB_SETUP.md` | Hướng dẫn setup training trên Colab |
| `OCR_METHOD_DECISION.md` | Quyết định phương pháp OCR đã thử |
| `resume_on_new_account.md` | Hướng dẫn resume training khi đổi Colab account |

## Ghi chú

- Root chỉ giữ `APP_ACTIVE_GENERAL_PLAN.md` và `APP_ACTIVE_DETAILED_PLAN.md` làm plan active.
- Các file test plan tạm thời hoặc execution board cũ không còn là nguồn ưu tiên.
