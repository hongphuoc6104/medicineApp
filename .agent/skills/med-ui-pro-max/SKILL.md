# Skill: Med UI Pro Max

This is the `medicineApp` adapter inspired by `ui-ux-pro-max-skill`.

## Design Direction For This Product

Preferred styles:
- Accessible & Ethical
- Inclusive Design
- Soft UI Evolution
- Minimalism & Swiss Style
- Nature Distilled

Avoid by default:
- cyberpunk
- vaporwave
- brutalism
- purple/pink AI gradients
- overly dark UI unless the project already uses it

## Screen-Level Expectations

- `Home`: dual state, no-plan vs has-plan
- `Scan`: clear guidance for glare/blur/cutoff
- `Edit Drugs`: low-friction correction before saving
- `Schedule`: high clarity, summary before save
- `History`: distinguish scan history from medication history
- `Settings`: controls must actually affect behavior

## Design System Pattern

Maintain:
- `design-system/MASTER.md`
- `design-system/pages/<screen>.md`

Page override wins over the master file.
