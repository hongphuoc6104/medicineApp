# Skill: Healthcare Accessibility

## Target User Considerations

- older users
- tired users
- users in a hurry during medication time
- users with low digital confidence

## UI Defaults

- clear light theme first
- generous spacing
- obvious buttons and states
- no decorative ambiguity for important actions
- readable contrast and hierarchy

## Critical Screens

- Home today summary
- scan guidance
- error and offline states
- medication confirmation buttons
