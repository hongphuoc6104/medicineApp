# Skill: Scan Session Flow

## Session Steps

1. start session
2. add images one by one
3. merge results
4. detect convergence
5. stop session and return merged view

## UI Contract Needs

- per-image status
- guidance after each upload
- merged drugs list
- convergence status

## Debug Focus

- repeated reject images
- unstable merged signatures
- new high-confidence drug appearing late in the sequence
